# Prism - 나만의 매력 스펙트럼을 발견하세요 ✨

**Prism**은 친구들과 가족들에게 익명으로 긍정적인 피드백을 받아 자신의 매력을 발견할 수 있는 웹 서비스입니다.

## 🌟 Prism이란?

Prism은 Gas 앱 모델을 기반으로 한 개인 매력 진단 서비스입니다. 
복잡한 설문이나 앱 설치 없이, 단순히 링크 하나를 공유하기만 하면 여러 사람으로부터 익명으로 긍정적인 피드백을 받을 수 있습니다. 

나를 보는 타인의 시선을 통해 내가 몰랐던 나만의 매력 포인트를 발견해보세요.

## 📱 사용 방법

### 1️⃣ 링크 생성하기
1. 메인 페이지에서 닉네임을 입력합니다
2. "링크 생성하기" 버튼을 클릭합니다
3. 고유한 투표 링크가 생성됩니다

### 2️⃣ 링크 공유하기
- 생성된 링크를 복사하여 친구, 가족, 동료들과 공유합니다
- 카카오톡, 인스타그램 등 다양한 채널을 통해 공유할 수 있습니다
- 링크를 받은 사람들은 별도의 가입이나 설치 없이 바로 피드백을 남길 수 있습니다

### 3️⃣ 피드백 받기
- 친구들이 여러 질문에 대해 익명으로 응답합니다
- 모든 질문은 100% 긍정적인 선택지로 구성되어 있어 편안하게 응답할 수 있습니다
- 응답은 완전히 익명으로 처리되어 솔직한 피드백을 받을 수 있습니다

### 4️⃣ 리포트 확인하기
- 최소 5명 이상의 응답이 모이면 리포트가 열립니다
- 태그 기반 분석을 통해 자신의 매력 포인트를 한눈에 확인할 수 있습니다
- 리포트는 실시간으로 업데이트되어 새로운 응답이 즉시 반영됩니다

## 💎 Prism의 주요 이점

### ✨ 완전한 익명성 보장
- 모든 응답은 완전히 익명으로 처리됩니다
- 응답자 정보가 저장되지 않아 더 솔직하고 진솔한 피드백을 받을 수 있습니다

### 🌈 100% 긍정적인 경험
- 모든 질문은 긍정적인 선택지로만 구성됩니다
- 부정적이거나 불편한 피드백 없이 즐겁게 자신의 매력을 발견할 수 있습니다

### 🔔 실시간 알림
- 새로운 응답이 들어올 때마다 실시간으로 알림을 받을 수 있습니다
- 리포트가 열릴 때까지의 진행 상황을 실시간으로 확인할 수 있습니다

### 🚀 간편한 사용
- 복잡한 가입 절차나 앱 설치가 필요 없습니다
- 링크 하나만 공유하면 바로 사용할 수 있습니다
- 모든 디바이스에서 바로 접근 가능한 웹 서비스입니다

### 🔒 안전하고 신뢰할 수 있는
- 개인정보 보호를 위한 보안 조치가 적용되어 있습니다
- Row Level Security(RLS)를 통해 데이터 안전성을 보장합니다

### 📈 바이럴 효과
- 피드백을 완료한 사람들이 쉽게 자신의 링크를 생성할 수 있습니다
- 자연스럽게 서비스가 확산되며 더 많은 피드백을 받을 수 있습니다

## 🎯 이런 분께 추천합니다

- **자신에 대해 궁금한 분**: 타인이 보는 나의 이미지가 궁금한 분
- **자신감을 키우고 싶은 분**: 긍정적인 피드백으로 자신감을 얻고 싶은 분  
- **특별한 날을 맞는 분**: 생일, 졸업 등 특별한 날 기념으로 선물하고 싶은 분
- **팀빌딩이 필요한 조직**: 팀원들 간의 긍정적인 피드백 문화를 만들고 싶은 팀
- **커뮤니티 운영자**: 커뮤니티 멤버들의 서로에 대한 긍정적 시선을 공유하고 싶은 분
